# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of Conduct.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A Korean translation is available at:
https://www.contributor-covenant.org/ko/version/2/1/code_of_conduct/

## Scope

This Code applies to all project spaces — the repository, issues, pull requests, discussions, and any channels where contributors represent the project — and to all contributors regardless of role.

## Reporting

To report a concern, contact the maintainer listed in [SECURITY.md](SECURITY.md). Reports are handled confidentially. The maintainer will review every report and respond as time allows.

## Enforcement

The maintainer follows the enforcement ladder described in the Contributor Covenant 2.1: correction, warning, temporary ban, permanent ban. Decisions are final for this repository.
