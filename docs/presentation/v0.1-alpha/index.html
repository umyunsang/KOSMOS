KOSMOS란?

한국 공공 API를 자연어로 만나는 시민 대화형 AI 플랫폼

미션

  • data.go.kr 10만 건 공공 데이터 + 5,000+ API를 단일 자연어 인터페이스로
  • 시민은 어느 부처가 어떤 API를 운영하는지 알 필요 없음
  • LLM이 의도 분석 → 적합한 공공 API 도구 자율 호출
  • 부처 횡단 복합 질의 처리

LLM Stack

  • 모델: K-EXAONE-236B-A23B (LG AI Research)
  • 서빙: FriendliAI Serverless (60 RPM Tier 1)
  • OpenAI-compatible native function calling

Full name

Korean Open Services Multi-agent Orchestration System

시민은 부처·API 경계를 학습하지 않습니다.
라우팅은 KOSMOS가 합니다.

Spec-Driven Workflow & Constitution

Spec-driven Workflow

→ /speckit-specify — 요구사항 스펙
→ /speckit-plan — 구현 계획 (vision.md 필수 참조)
→ /speckit-tasks — 태스크 분해
→ /speckit-analyze — Constitution 컴플라이언스
→ /speckit-taskstoissues — GitHub Sub-Issues API v2
→ /speckit-implement — Agent Teams 병렬 실행
→ PR (Closes #EPIC only)

Constitution 6 Principles

  • I. Reference-Driven Design (vision.md + .references/ 인용)
  • II. Fail-Closed Security (기본값 = 거부)
  • III. Pydantic v2 Strict Typing (no Any)
  • IV. Gov API Compliance (CI live API 호출 금지)
  • V. Policy Alignment (PIPA §26 + AI Action Plan)
  • VI. Deferred-Work Accountability (비이행 Epic 추적)
