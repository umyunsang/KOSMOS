# Tool Adapter: `nfa_emergency_info_service`

**Provider**: 소방청 (National Fire Agency, NFA)
**Tool ID**: `nfa_emergency_info_service`
**Status**: Interface-only stub — live HTTP implementation pending Epic #16 / #20 (Layer 3 auth gate)

---

## Overview

Historical, anonymized EMS activity statistics from the NFA `EmergencyInformationService`
(data.go.kr catalog ID 15099423, provider code `1661000`). Returns dispatch statistics
by region, fire station, and report year-month.

**This is NOT a real-time 119 dispatch tool.** It does not return current-incident data
or facility locator results. For nearest 119 station lookup, see `nfa_safety_center_lookup`
(deferred to #934).

---

## Endpoint

```
GET https://apis.data.go.kr/1661000/EmergencyInformationService/{operation}
```

**Authentication**: `serviceKey` query parameter (URL-encoded).

---

## Available Operations

| Enum value | Operation path | Korean name |
|---|---|---|
| `activity` (default) | `getEmgencyActivityInfo` | 구급활동정보 목록조회 |
| `transfer` | `getEmgPatientTransferInfo` | 구급환자이송정보 목록조회 |
| `condition` | `getEmgPatientConditionInfo` | 구급환자상태정보 목록조회 |
| `firstaid` | `getEmgPatientFirstaidInfo` | 구급환자응급처치정보 목록조회 |
| `vehicle_dispatch` | `getEmgVehicleDispatchInfo` | 구급차량출동정보 목록조회 |
| `vehicle_info` | `getEmgVehicleInfo` | 구급차량정보 목록조회 |

The default `activity` operation returns: dispatch distance, patient symptoms,
patient age bracket, and crew qualifications — the fields most useful for
citizen-facing conversational guidance.

## Interface-Only Status

`handle()` raises `Layer3GateViolation` unconditionally. The Layer 3 auth-gate in
`executor.invoke()` short-circuits `requires_auth=True` calls before `handle()` is
reached (FR-025, FR-026, SC-006). The gate fires first and returns
`LookupError(reason="auth_required")` with zero upstream HTTP calls.

Live implementation will be added when the Layer 3 auth gate ships (Epic #16 / #20).
