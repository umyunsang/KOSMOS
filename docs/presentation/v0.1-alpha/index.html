"엄마가 갑자기 쓰러지셨어요. 119 부르고 가장 가까운 응급실 어디예요?"
3 부처 동시 대응 (NFA119 · HIRA · KMA): 119 자동 호출 가이드 + NMC 가용 병상 실시간 조회 + KMA 낙상위험 기상경보
→ "119 출동 요청됨. 가장 가까운 응급실: 강남세브란스 8분, 가용병상 3. 빙판 주의 — 도로 결빙 경보 발효 중."
"회사가 갑자기 망했어요. 실업급여·직업훈련·임대주택 우선순위까지 알아봐야 해요."
4 부처 동시 대응 (고용부 · 워크넷 · 마이데이터 · LH): 실업급여 자격 + 직업훈련 매칭 + 소득 자동 인증 + LH 가산점
→ "실업급여 월 184만원 × 240일 신청 가능. 훈련 5종 추천. LH 임대 우선순위 +20점. 14일 내 고용센터 방문."
"전세사기 당했어요. 어떻게 해야 해요?"
4 부처 동시 대응 (HUG · 법무부 · 법률구조공단 · 경찰청): 보증보험 청구 + 임대차분쟁조정 + 무료 상담 예약 + 형사 신고
→ "HUG 청구 → 조정신청 → 형사 신고 3트랙 동시 권고. 법률구조공단 무료 상담 예약: 내일 14시."

Chapter 1 — 프로젝트 개요

6-Layer Architecture

1
Query Engine
while(True) 도구 루프 — 시민 요청 완전 해결까지 비동기 제너레이터
2
Tool System
data.go.kr API 어댑터 스키마-구동 레지스트리 · BM25+dense 하이브리드 검색
3
Permission Pipeline
시민 인증 + PIPA 개인정보 보호 게이트 · bypass-immune 3-layer
4
Agent Swarms
부처 전문 에이전트 오케스트레이터 협업 · mailbox IPC 충돌 복구
5
Context Assembly
LLM이 매 턴 받는 3-tier 컨텍스트 · 프롬프트 캐시 파티션
