서비스 사용
서비스 Key 발급 및 활용

서비스 Key발급(인증신청 및 발급)
(1) 공공데이터포털(http://data.go.kr) 활용자
- 활용신청하고자 하는 API를 검색하여 클릭


- 활용 신청 하고자 하는 서비스 항목에 활용신청 버튼 클릭


- 해당되는 내용을 기재하고 이용하고자 하는 상세기능을 체크한뒤 가장 아래 쪽 파란 신청 버튼을 클릭



- 신청이 완료되면 아래와 같은 메시지 박스가 출력된다



- 신청이 완료되면 신청목록이 조회되어 출력된다



- 왼쪽의 메뉴에 인증키 발급현황을 클릭하면 인증키 확인이 가능하다















서비스 인증키 활용
(1) 제공기관 활용자
공공데이터포털 페이지(https://www.data.go.kr) 접속후 아이디/비밀번호, 인증서를 사용하여 로그인한다.


- 마이페이지에서 개발계정 신청 내역을 조회한다.







- 마이페이지에서 서비스명을 클릭하면 개발계정 상세보기 화면으로 이동한다.



목록화면에서 상세기능정보 실행 버튼을 클릭시, 요청변수(Request Parameter) 내용을 확인 할 수 있다. 미리보기 버튼을 클릭시 아래와 같은 오픈API 결과값을 받을 수 있다.
