Chapter 01 · Overview

한국 공공 API를 자연어로 만나는 시민 대화형 AI 플랫폼

시민은 어느 부처가 어떤 API를 운영하는지 알 필요 없이, 자연어 한 줄로 5,000+ 한국 공공 API를 호출할 수 있습니다.

01
CORE PROPOSITION 단일 자연어 인터페이스로 5,000+ API 자율 호출
  • data.go.kr 10만+ 데이터셋 + 5,000+ API — 자연어 한 줄로 통합.
  • K-EXAONE-236B-A23B (LG AI Research) — 한국어 특화 LLM 도구 루프.
  • FriendliAI Serverless — 60 RPM Tier 1 · OpenAI-compatible native function calling.
  • 4 primitive 동사 — lookup · submit · verify · subscribe.
FULL NAME KOSMOS = Korean Open Services Multi-agent Orchestration System
  • 학생 포트폴리오 · KSC 2026 출품 · Apache-2.0 오픈소스.
  • 시민은 부처·API 경계를 학습하지 않습니다. 라우팅은 KOSMOS가 합니다.
  • Not affiliated with LG AI Research or the Korean Government.
PUBLIC APIS5,000+data.go.kr 등록 오픈 API
DATASETS100K+공공 데이터셋
PRIMITIVES4lookup · submit · verify · subscribe
MINISTRIES7KOROAD · KMA · HIRA · NMC · NFA · MOHW · MyData
VISION · 정부 공식 정책 부합

“DX는 충분하다. 지금 필요한 것은 분산된 공공 인프라를 단일 지능으로 묶는 AX 레이어다.”

Chapter 01 · Citizen Scenarios

시민 사용 시나리오 — 부처 횡단 긴급 대응

시민이 처한 위기 상황 3가지. 단일 자연어 입력 → KOSMOS가 3-4 부처 API를 동시 호출 → 통합 답변 + 다음 행동 권고.

01
"엄마가 갑자기 쓰러지셨어요. 119 부르고 가장 가까운 응급실 어디예요?"
3-4 부처 동시 (NFA119 · HIRA · NMC · KMA) — 119 자동 호출 가이드 + NMC 가용 병상 실시간 + KMA 도로 결빙 경보 → "강남세브란스 8분, 가용병상 3. 빙판 주의 — 도로 결빙 경보 발효 중."
"회사가 갑자기 망했어요. 실업급여·직업훈련·임대주택 우선순위까지 알아봐야 해요."
4 부처 동시 (고용부 · 워크넷 · 마이데이터 · LH) — 실업급여 자격 + 직업훈련 매칭 + 소득 자동 인증 + LH 가산점 → "월 184만원 × 240일. 훈련 5종 추천. LH 임대 우선순위 +20점. 14일 내 고용센터 방문."
"전세사기 당했어요. 어떻게 해야 해요?"
4 부처 동시 (HUG · 법무부 · 법률구조공단 · 경찰청) — 보증보험 청구 + 임대차분쟁조정 + 무료 상담 예약 + 형사 신고 → "HUG → 조정 → 형사 신고 3트랙 동시 권고. 법률구조공단 무료 상담 내일 14시."
다음 3 슬라이드 — 위 3 시나리오를 ink-testing-library로 캡쳐한 실제 KOSMOS TUI 화면 (5 step × 3 = 15 frame).
